# שאלון אדם רגיש מאוד

גרסה מקוונת של שאלון "אדם רגיש מאוד" מאת ד"ר איליין ארון.

## קבצים

- `index.html` - דף השאלון הראשי.
- `styles.css` - עיצוב התצוגה.
- `script.js` - הלוגיקה לחישוב תוצאה.

## שימוש

1. פתח את `index.html` בדפדפן.
2. מסמן "כן" או "לא" לכל שאלה.
3. לחץ על הכפתור כדי לראות את התוצאה.
